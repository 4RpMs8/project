آموزش ساخت سایت در GitHub

توسط سید محمد شایان رضوی

۱. ایجاد مخزن جدید در GitHub

برای شروع، به GitHub وارد شوید و یک مخزن جدید ایجاد کنید.

            ۱. به صفحه GitHub بروید و وارد حساب کاربری خود شوید.
            ۲. بر روی دکمه "New" در بالای صفحه کلیک کنید.
            ۳. نام مخزن خود را وارد کنید و توضیحی برای آن بنویسید.
            ۴. نوع مخزن را عمومی (Public) یا خصوصی (Private) انتخاب کنید.
            ۵. بر روی دکمه "Create repository" کلیک کنید.

۲. کلون کردن مخزن به کامپیوتر

پس از ایجاد مخزن، باید آن را به کامپیوتر خود کلون کنید.

            ۱. لینک HTTPS یا SSH مخزن را کپی کنید.
            ۲. یک ترمینال یا خط فرمان باز کنید و دستور زیر را اجرا کنید:
            git clone <URL>
            (به جای <URL>، لینک کپی شده را وارد کنید)

۳. انتقال فایل‌ها به پوشه مخزن

فایل‌های HTML، CSS و تصاویر خود را به پوشه مخزن منتقل کنید.

            ۱. فایل‌های HTML، CSS و تصاویر خود را به پوشه مخزن که به کامپیوتر خود کلون کرده‌اید منتقل کنید.
            ۲. تغییرات خود را با دستور زیر به مخزن اضافه کنید:
            git add .
            ۳. تغییرات خود را کامیت کنید:
            git commit -m "انتقال فایل‌ها به پوشه مخزن"

۴. افزودن فایل HTML و CSS به مخزن

فایل HTML و CSS خود را ایجاد و به مخزن اضافه کنید.

            ۱. یک فایل HTML با نام index.html ایجاد کنید و کد HTML خود را در آن بنویسید.
            ۲. یک فایل CSS با نام styles.css ایجاد کنید و کد CSS خود را در آن بنویسید.
            ۳. تغییرات خود را با دستور زیر به مخزن اضافه کنید:
            git add .
            git commit -m "افزودن فایل HTML و CSS"
            git push origin main

۵. افزودن تصاویر و منابع به مخزن

تصاویر و منابع مورد نیاز خود را به مخزن اضافه کنید.

            ۱. تصاویر و منابع مورد نیاز خود را به پوشه مخزن اضافه کنید.
            ۲. تغییرات خود را با دستور زیر به مخزن اضافه کنید:
            git add .
            git commit -m "افزودن تصاویر و منابع"
            git push origin main

۶. پیاده‌سازی بازی حدس عدد در وب‌سایت

بازی حدس عدد را به وب‌سایت خود اضافه کنید.

            ۱. کد HTML و JavaScript بازی حدس عدد را به فایل HTML خود اضافه کنید.
            ۲. تغییرات خود را با دستور زیر به مخزن اضافه کنید:
            git add .
            git commit -m "افزودن بازی حدس عدد"
            git push origin main
